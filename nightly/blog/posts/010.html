possibly moving(?)

neocities is great. it provides an easy way to edit files and more! but some things that ive realized is that auth's RNG is open source and anyone can contribute to it, and making community made changes is gonna be potentially a nightmare.

i need a hosting platform thats merge-deploy-and-go. so ive settled on choosing Cloudflare Pages.

this might sound catastrophic, but could counter a lot of issues in the future. heres why auth's RNG is moving to Cloudflare Pages:

  • easier to change and deploy

    • i have neocities's save button, what more do i need? again, auth's RNG is open source and manually changing files back on neocities and deploy is a mightmare. i want it so as soon as i merge or accept the pull request, those changes are reflected back onto the site automatically.


  • faster load times

    • with cloudflare's CDN, the site is gonna load kilometers and kilometers faster. neocities's anycast CDN is good, but cloudflare's is a bit more better


  • less restrictions

    • cloudflare doesnt have a strict CORS and CSP as neocities, allowing ranked mode to be real!!! also, cloudflare allows me to set a custom domain FOR FREE (neocities blocks it with a subscription) which means authsrng.xyz soon????

overall i hope im not actively ruining auth's RNG, but the switch is to combat any potential future issues.

once i make the cloudflare pages site, ill announce it. you'll have to download/copy your save file and import it into the new main site to prevent any data loss. if you care about your data, move along.

but what are you gonna do with the neocities site?

ill just make it redirect to the new site. no questions asked.
