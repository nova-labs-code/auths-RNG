why page system?

you probably hate the current page system in the game and you're probably asking me, why add it if it's "useless"? the reason im making this post is that many, many people really don't like the page system and i wanna address that.

well, it's main goal is to help beginners get a grip on the game without feeling overwhelmed. before i implemented the page system, i showed auth's RNG to a few of my cousins that haven't played, nor knew about auth's RNG before. they spent a total of 2 to 5 minutes trying to get a grip on the game because they were overwhelmed and didn't know what to do.

i figured "oh its because literally everything is on a single page" so i added the page system in.

another reason for adding the page system is that with the single page design... it didn't really feel like an actual web game and just a random website that just has a little minigame in it. it icked me a lot when i was testing because it really didn't feel like a web game. sure, sure, you could say that's just bias, but again im doing what i believe is best for the game (i still listen to you guys though)

when you guys were complaining about the page system, i was quite pissed that you guys couldn't give enough feedback as to why it sucked! all i got was "its just useless" and "i prefer the old one". the only actual feedback i got from this was "it takes forever to go from somewhere to somewhere else". I WANNA MAKE THE GAME BETTER! and if you want the game to be better, just give me better feedback next time? alright? thanks
